# Monitor Deportivo — Dashboard en Dash - AxisFit

## ¿Qué hace esta app?
AxisFit es una plataforma de monitorización en tiempo real que analiza la postura y el movimiento mediante sensores inteligentes. Permite evaluar la calidad técnica, detectar desalineaciones y medir la carga biomecánica durante entrenamientos o actividades diarias, ayudando a prevenir lesiones y optimizar el rendimiento.

## Tecnologías usadas
- Python + Dash + Plotly
- Sensores IMU
- Base de datos integrada para análisis y seguimiento
